高市総務相の靖国参拝「避けるべきだ」　公明・山口氏

2014年10月15日11時35分

  • シェア 0
  • ツイート 0
  • ブックマーク 0
  • メール
  • 印刷
[PR]

　公明党の山口那津男代表は１５日のラジオ日本の番組で、高市早苗総務相が靖国神社参拝の意向を示したことについて、日中関係などを念頭に「外交的な課題をつくるのは避けるべきだ」と語った。

　日中両国は、１１月に北京で開かれるアジア太平洋経済協力会議（ＡＰＥＣ）での首脳会談の実現に向けて水面下の調整を進めている。山口氏は「それに水を差すようなことを避けて、環境を整える役割をみんなで果たすということが大事だ」とも述べた。

　また、日米両政府が年内の改定を目指している「防衛協力のための指針（ガイドライン）」について「法体系を整えるのには少し時間がかかる。時期に必ずしもこだわらない」と話した。
